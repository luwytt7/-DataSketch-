# DataSketch: Transforming Data into Art

**DataSketch** is a Python project that transforms datasets into artistic visualizations. The goal is to combine data visualization with generative art, creating unique sketches from data.

## Features
- Generates artistic waveforms from any dataset.
- Saves the result as a `.png` image.
- Fully customizable (colors, sizes, data).
